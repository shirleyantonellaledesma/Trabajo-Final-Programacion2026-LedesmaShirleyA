# Trabajo Final Integrador - Programación

Este repositorio contiene la resolución del Trabajo Final Integrador de la materia **Programación**, correspondiente a la **Tecnicatura en Telecomunicaciones**.

## Autor
* **Shirley Antonella Ledesma**

## Descripción del Proyecto
Este proyecto reúne las actividades prácticas solicitadas durante la cursada. El repositorio incluye el desarrollo completo de los 10 ejercicios propuestos, todos implementados en **PSeInt**, abarcando desde lógica básica y estructuras de control, hasta manipulación de cadenas, matrices y algoritmos de redes.

## Estructura del Repositorio

| Ejercicio | Descripción |
| :--- | :--- |
| `EJERCICIO1_ContadorVocales.psc` | Contador de vocales en frases. |
| `EJERCICIO2_Invertir.psc` | Inversión de cadenas de texto. |
| `EJERCICIO3_SumaDeMatrices.psc` | Suma de matrices de 2x2. |
| `EJERCICIO4_Triangulo.psc` | Cálculo de área de triángulo. |
| `EJERCICIO5_TablaMultiplicar.psc` | Tabla de multiplicar hasta x20. |
| `EJERCICIO6_Piramide.psc` | Generación de pirámide numérica. |
| `EJERCICIO7_Interes.psc` | Cálculo de interés simple. |
| `EJERCICIO8_AdivinarNumero.psc` | Juego de adivinanza con pistas. |
| `EJERCICIO9_RedesYComunicacion.psc` | Modelado de red direccionada de 8 nodos. |
| `EJERCICIO10_Promedio.psc` | Promedio con corte por número negativo. |

## Proceso de Trabajo
He desarrollado cada ejercicio de manera individual, asegurándome de cumplir estrictamente con las consignas de la cátedra. Durante el proceso, puse especial énfasis en:
* **Sintaxis estricta:** Uso correcto de puntos y coma (`;`) y estructuras solicitadas.
* **Lógica:** Verificación de cada algoritmo mediante pruebas de escritorio con distintos conjuntos de datos.
* **Documentación:** Comentado claro del código para facilitar su lectura y corrección.
